Instruction: how to read the plots

The following plots are the same present in the end of each analysis, reported here to have a better overview.

Each plot is composed of a heatmap with an enrichement score (from 0 to 1) for each marker gene set in each cell cluster. The cell cluster number is indicated for each row with roman number in the rightmost column. For the marker score, if a gene is considered a marker for more than one group, the score is proportionally reduced. The last column of the heatmap contains the score for the ISG set of genes. On the left there is a dendrogram evaluated using a distance (cosine) over all COTAN uniform clusters and considering, to estimate the distance, the whole transcriptome. The green bar plot between the dendrogram and the heatmap shows the UMI detection efficiency (UDE) mean. This score is the RNA extraction efficiency mean for the cell cluster. This can be usefull because if we observe a very low UDE and no clear marker group enrichment or there is an enrichment for more than one marker set (and very different), is it possible that the cluster is formed by just background signal. The right orange bar plot shows the cluster cell number.
